奇遇日誌

萬物皆有故事，等你停下來聽。

世界上每個角落，都住著故事化成的小生命——小遇。牠們不躲人，只是大家走得太快。

你手上這本會發光的旅行手札，是一位探索者留下的。停下來，把鏡頭對準那個角落——認識一隻小遇、聽懂牠的故事，牠就會把一個小小的自己託付給你，住進日誌的新一頁。

新來的探索者你好！用鏡頭對準會發光的圖騰試試。　——艾可

測試須知

  1. 相機功能需要 HTTPS 或 localhost 環境（手機實測請部署到 GitHub Pages）。
  2. 用另一台裝置開啟「測試用圖騰」頁面，或將它列印出來。
  3. 點「出發探索」並允許相機權限，將鏡頭對準圖騰圖案。
